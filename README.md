### Описание проекта
Компьютер загадывает целое число от 1 до 100, и нам нужно его угадать. Под «угадать» подразумевается написать программу, которая угадывает число за минимальное количество попыток.

:arrow_up: [к оглавлению](https://github.com/Bakhty-s/game-guess-number#readme)

### Какой кейс решаем?
Нужно написать программу, которая угадывает число за минимальное количество попыток.

**Условия соревнования:**  
- Компьютер загадывает целое число от 1 до 100.  
- Алгоритм угадывает число, используя информацию о том, больше или меньше случайное число нужного.

**Метрика качества**  
Результаты оцениваются по среднему количеству попыток при 1000 повторений.

**Что практикуем**  
Учимся писать хороший код на Python, работать с Git и GitHub, оформлять проекты.

### Краткая информация о данных
В проекте используются случайно сгенерированные числа от 1 до 100.

### Этапы работы над проектом
1. Написана базовая версия игры с случайным угадыванием.  
2. Улучшен алгоритм с использованием бинарного поиска.  
3. Проведено тестирование алгоритма.

### Результат
Программа угадывает число в среднем за X попыток.
